HTML

HTML stands for HyperText Markup Language, which might not say much for everyone but basically put HTML is the basis of all web language used today. Since the birth of HTML 1995 new languages and adaptations have emerged such as CSS and Java. With HTML code anyone can create a range of different websites containing from a little to a lot of information. For example everything you see on this page has been done with simple HTML code. The use of very little CSS code also known as style code has been utilized in a linked document.
